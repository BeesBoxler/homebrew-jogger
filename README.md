# homebrew-jogger

Homebrew tap for Jogger macOS app.

## Install

```bash
brew tap beesboxler/jogger
brew install --cask jogger-macos
```

## Upgrade

```bash
brew update
brew upgrade --cask jogger-macos
```

## Maintainers

Update cask for a new release:

```bash
scripts/update-cask.sh 0.3.4
```

Then commit and push.
